Hermit Quiz Arena

Beginner-friendly deployment guide for Render.

What you need

  • A Telegram account
  • A GitHub account
  • A Render account
  • The extracted Hermit Quiz Arena folder

You do not need to install Python, Node.js or coding software.

1

Create your Telegram bot

  1. Open @BotFather in Telegram.
  2. Send /newbot.
  3. Choose a bot name and username.
  4. Copy the bot token and keep it private.
2

Upload the project to GitHub

  1. Create a new empty GitHub repository.
  2. Upload every file and folder inside the extracted project folder.
  3. Do not upload the ZIP itself.
  4. Do not create or upload .env or node_modules.
3

Create the Render service

  1. Sign in to Render and connect GitHub.
  2. Click New +, then Blueprint.
  3. Select your Hermit Quiz Arena repository.
  4. Render will read render.yaml.
  5. Enter the requested secret values and create the Blueprint.
4

Add the environment values

NameValue
BOT_TOKENYour private BotFather token
BOT_USERNAMEYour bot username without @
ADMIN_PASSWORDA strong password you will remember
PUBLIC_URLUse https://placeholder.onrender.com at first

Render generates APP_SECRET automatically.

5

Replace the placeholder URL

  1. Wait until the service status is Live.
  2. Copy its real address, such as https://hermit-quiz-arena.onrender.com.
  3. Open the service's Environment page.
  4. Replace PUBLIC_URL with the real address.
  5. Save the changes and redeploy.
6

Register the Telegram Mini App

  1. Return to @BotFather and send /newapp.
  2. Select your bot.
  3. When asked for the Web App URL, paste your Render HTTPS address.
  4. Choose a short name such as arena.
  5. In Render Environment, set MINI_APP_SHORT_NAME to that exact short name, then redeploy.
7

Connect your Telegram group

  1. Add the bot to your Telegram group.
  2. Send /quizsetup@YourBotUsername in the group.
8

Create your first quiz

  1. Open https://YOUR-RENDER-ADDRESS/admin.
  2. Sign in with your ADMIN_PASSWORD.
  3. Import samples/questions_template.csv.
  4. Create a live quiz, announce it in the group and start it.

Important free-plan limitation

Render Free uses temporary local storage. Imported quizzes and results can disappear after a restart, redeployment or service shutdown. It is suitable for testing. Permanent storage requires a paid Render disk or a hosted database.

Never share these publicly

  • BOT_TOKEN
  • ADMIN_PASSWORD
  • APP_SECRET

Keep them only in Render Environment variables.
